Module: hooks/useAuth

Custom hook para acceder al contexto de autenticación
Properties:
Name Type Description
user Object | null Usuario autenticado con sus datos (uid, email, name, role)
isLogged boolean Indica si hay un usuario autenticado
role string | null Rol del usuario ('user' o 'admin')
loading boolean Estado de carga de operaciones de autenticación
initialLoading boolean Estado de carga inicial al verificar la sesión
authError string | null Mensaje de error de autenticación
setAuthError function Función para establecer errores de autenticación
register function Función para registrar un nuevo usuario
adminCreateUser function Función para que admin cree usuarios
login function Función para iniciar sesión
logout function Función para cerrar sesión
resetPassword function Función para enviar email de recuperación
changePassword function Función para cambiar contraseña

const { user, isLogged, login, logout } = useAuth();
